All the files in this directory are generated by the configuration script
and can later be edited to finalize the configuration.

ccc-compilers.cfg
 This file has a listing of all compilers available on the platform
 including their respective CCC COMPILER IDs ...

ccc-glob-flags.<COMPILER>.cfg
 This file has a list of global compiler flags for iterative compilation.

ccc-env.sh
 This file has environment variables. 
 Should be executed before using CCC framework.
